# Triangle Checker

#### This is a simple Ruby app the checks to see if 3 given numbers could make a triangle. If it is a triangle, the program checks to see what kind; equalateral, isosceles, or scalene

Specs:
* Is it a triangle?
  **Input:** 2, 3, 8
  **Output:** False
* What kind of triangle?
  **Input:** 2, 2, 3
  **Output:** Isosceles
